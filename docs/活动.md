一、最高价值的一类：“发现网站的网站”

这是你真正想要的东西。

1️⃣ Hackathon / 比赛聚合型（强推荐）
国际（英文为主）

这些网站的核心价值不是“活动”，而是帮你发现主办方和平台

Hackathon.com
👉 全球 Hackathon 索引
能看到：

主办组织

官方活动站

合作平台
👉 非常适合反向扒“谁在长期办活动”

Devpost – Hackathons
👉 黑客松发布 + 项目托管
价值点：

每个 Hackathon 页面都会链接主办方官网

你可以直接抽取「主办机构名单」

Major League Hacking (MLH)
👉 学生向高奖金黑客松组织
价值点：

MLH 的合作伙伴 = 一整批长期出钱的公司

反查 sponsors，是一条金矿线索

Challenge.gov（美国政府）
👉 美国政府官方竞赛平台
价值点：

能发现大量政府部门、研究机构、基金会官网

奖金极高，门槛偏高
